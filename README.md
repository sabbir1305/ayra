# Ayra Properties Ltd. Website

A responsive and attractive website for Ayra Properties Ltd., a premier land selling company.

## Features

- **Responsive Design**: Built with Bootstrap 5 for optimal viewing on all devices
- **Modern UI**: Clean, professional design with attractive gradients and animations
- **Sections**:
  - Hero section with company introduction
  - About Us section
  - Previous Projects carousel
  - Running Projects carousel
  - Contact Us form and information
- **Smooth Scrolling**: Navigation links with smooth scroll behavior
- **Interactive Elements**: Bootstrap carousels and form handling

## Technologies Used

- HTML5
- CSS3
- Bootstrap 5.3.0
- JavaScript (ES6)
- Google Fonts (Poppins)

## Project Structure

ayra-properties/

## How to Run

1. Clone or download the project files
2. Open `index.html` in a web browser
3. For local development with a server (recommended):
   - If you have Python installed: `python -m http.server 8000`
   - If you have Node.js: `npx serve .`
   - Then open `http://localhost:8000` in your browser

## Customization

- **Images**: Replace placeholder images in `index.html` with your actual project images
- **Content**: Update text content in `index.html` to match your company's information
- **Styling**: Modify `css/custom.css` to change colors, fonts, or layout
- **Contact Information**: Update the contact details in the Contact Us section
